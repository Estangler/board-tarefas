# 📌 Board de Tarefas

Aplicação de gerenciamento de tarefas inspirada em boards Kanban, desenvolvida com **React**, **TypeScript**, **Context API** e **useReducer**.

O projeto foi criado com foco em praticar arquitetura frontend moderna, gerenciamento de estado global, componentização e organização escalável de aplicações React.

# 🎯 Objetivos do projeto

Este projeto foi desenvolvido para praticar:

* Componentização no React
* Gerenciamento de estado global
* Fluxo de ações com reducers
* Organização escalável de pastas
* Manipulação de formulários
* Validação de inputs
* Estruturação de UI estilo Kanban
* Boas práticas com TypeScript

---

# 🧠 Aprendizados

Durante o desenvolvimento, pratiquei conceitos importantes como:

* Separação de responsabilidades
* Criação de componentes reutilizáveis
* Fluxo de estado centralizado
* Estruturação de reducers e actions
* Melhor organização de código em aplicações React
